## Core Concept

Progressive Web App guiding users through 3 levels of practical AI mastery using **only natural language** (no prompting tips, no Zapier):

- **Beginner**: ChatGPT beyond search ("Plan my groceries from receipt photo")
    
- **Intermediate**: Simple delegation ("Clean my Downloads folder")
    
- **Advanced**: Full agents (Claude Cowork → "Receipts to expense report", Perplexity Computer → "Research + dashboard")
    

**Key Insight**: Abundant TikTok/Reels/YouTube Shorts already demonstrate this exact progression—your app just sequences them perfectly.

## User Experience Flow

1. **Discovery**: Shared URL (`yourapp.vercel.app`) → no app store needed
    
2. **Landing**: 15s teaser video → "Start Beginner Path"
    
3. **Learning**: 3 cards/level → embedded social videos → "Try it" opens ChatGPT/Claude with prefilled natural prompt → progress bar → next level
    
4. **PWA Magic**: Auto-adds home screen icon (iOS Safari/Chrome) → feels native, offline progress
    
5. **Completion**: "You're now an AI manager!" → live demo playground → share badge
    

**Time**: 20 minutes total. Gamified with streaks/confetti.
